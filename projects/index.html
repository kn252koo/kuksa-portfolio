Навчальні проєкти

1. Проєкт "Personal Portfolio Page"

Опис: Створення персональної веб-сторінки для презентації власних навичок та виконаних робіт. Проєкт включає структурування контенту, створення навігації та інтеграцію медіафайлів. Головна увага приділена чистоті коду та логічній ієрархії заголовків.

Використані технології: * Markdown: для написання та форматування текстового контенту. * Git: для відстеження версій та збереження історії змін. * GitHub: для хостингу репозиторію та публікації сторінки.

2. Проєкт "Data Tracker & Analyzer"

Опис: Розробка системи для обліку та візуалізації щоденних витрат або навчального прогресу. Проєкт передбачає створення бази даних у табличному вигляді, розрахунок автоматичних показників та створення графіків для аналізу ефективності.

Використані технології: * MS Excel: створення таблиць, використання складних формул (VLOOKUP, IF) та зведених таблиць. * Зображення: підготовка візуальних звітів для документації. * Markdown: для створення супровідної документації (README).

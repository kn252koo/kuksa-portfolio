#Ключові компетенції
* **Робота з Git та GitHub**: ініціалізація репозиторіїв, додавання змін (`git add`), створення комітів (`git commit`) та робота з віддаленими серверами.
* **Розмітка Markdown**: створення структурованих документів, оформлення списків, робота з посиланнями та інтеграція зображень.
* **MS Excel та аналіз даних**: побудова таблиць, використання формул для розрахунків та візуалізація даних.
* **Робота з терміналом**: використання командного рядка для навігації по файловій системі та виконання базових операцій розробника.
* **Основи веб-технологій**: розуміння структури проектів та принципів створення статичного контенту.
